# Content Enhancer Agent Examples

LangGraph pipeline that enhances and improves existing text — fixing grammar, improving word choice, and strengthening clarity.

## Enhance Content

```bash
npx tsx playground/content_enhancer_agent/index.ts -i "The company have been growing alot since last year and there revenue is up."
npx tsx playground/content_enhancer_agent/index.ts -i "Me and him went to the store for buying some stuffs."
npx tsx playground/content_enhancer_agent/index.ts -i "Technology is changing fast and it affects many things in our daily life in many ways."
npx tsx playground/content_enhancer_agent/index.ts -i "There are many reasons why people should exercise more because it is good for health and also helps with mental wellbeing and stuff like that."
```
